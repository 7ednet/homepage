蚂蚁短链 (Mayi.ws) 是一个域名缩短服务，它可以帮助用户将长而复杂的网址缩短为短、简洁的形式，方便用户在社交媒体、即时通讯工具或其他场合进行分享。使用蚂蚁短链可以使原本难以记忆或分享的长网址变得简单易用，不跟踪、不记录。此外，蚂蚁短链还提供了定制短链接的功能，让用户可以创建更具品牌特色的短链接。

1/ HOW TO USE ​

🐜

立即前往注册使用蚂蚁短链 Mayi.ws

2/Effect ​

例如在 macOS、Linux（包括 WSL）使用 n 安装 Node.js，需输入一长串字符：

bash
bash <(curl -L https://raw.githubusercontent.com/tj/n/master/bin/n) lts

现可使用蚂蚁短链缩短的链接，很方便的就可以输入了：

bash
bash <(curl -L mayi.ws/n) lts
