# RSS 一站式服务

<!-- toc -->

:RSS 一站式服务包含 RSSHub 后端、Miniflux 阅读器、Miniflux 第三方客户端三个服务。


## RSSHub 后端

* 网址: https://rss.con.sh

使用方式见 RSSHub 官方文档。

## Miniflux 阅读器

* 网址: https://reader.con.sh

使用 Google 账号登陆即可使用，在设置页面可更改时区、语言等内容。

::: info :secret:
若您需要更优美的使用界面，可使用下方 Miniflux 第三方客户端，需要在 API 密钥页面生成密钥以供连接。
:::

## Miniflux 第三方客户端

* 网址： https://r.con.sh

使用方法：

* `Server address`: 填写 https://reader.con.sh
* `API Token`: 填写 API 密钥页面生成的密钥。
