# Unsend.de 临时邮箱服务

<!-- toc -->

一个临时邮箱服务。

## Unsend.de 网站

* 网址: https://unsend.de

使用 Github 登录或输入用户名密码进行注册。

## Telegram 机器人

* 网址: https://t.me/unsenddebot

将机器人给出的 Webhook 地址填入 https://unsend.de/profile 相关表单中即可在 Unsend TG 机器人收取新邮件通知。

在 https://unsend.de/profile 生成 API Key，并在 TG 机器人进行设置，即可在 TG 机器人进行创建临时邮箱等操作。
