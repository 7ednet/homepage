# Commento.net 公共评论系统

<!-- toc -->

Commento.net 是一个基于 Commento 的公共评论系统，仅需注册引用即可在您自己的静态博客为用户提供评论系统，支持 Github 登录、Google 登录、匿名评论。

Commento 有官方托管服务，叫价 $99 美元/年，现免费提供给您。

<a href="https://commento.net" target="_blank" style="background-color: var(--theme-color); color: #fff; text-decoration: none; padding: 10px;">👌 立即注册 Commento.net</a>

## 1/ 使用方式

前往 Commento.net 注册，并添加网址，然后在您的博客中适当位置引用以下代码即可：

```html
<script defer src="https://commento.net/js/commento.js"></script>
<div id="commento"></div>
```

## 2/ 注意

添加网站时，网站地址使用 `%.example.com` 可以匹配 `example.com` 和所有子域，否则只能匹配填写的域名。
