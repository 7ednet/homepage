仅需在 Github 资源网址后加个 S 即可在中国大陆地区畅快访问。

## 1/ HOW TO USE

将原网址

```html
raw.githubusercontent.com
```

加个 `S`，变成

```html
raw.githubusercontentS.com
```

即可，例如：

```html
https://raw.githubusercontents.com/imagehosted/images/master/2019/06/20190612115457.png
```
